## Overview

**claude-code-plugin-mcp** is an open-source integration for AI-powered developer tools and code intelligence.

TypeScript plugin to extend Claude Code with custom MCP servers for local file manipulation.

Topics: `claude-code` `mcp` `devtools`

## Features

- 🚀 Production-ready integration targeting ai devtools use cases
- 🔧 Easy to integrate into existing projects
- 📦 Zero-configuration defaults with full customization support
- 🧪 Comprehensive test coverage
- 📖 Well-documented API with examples
